# Simplify Git Demo

Demo project để thực hành clean git history.

## Branches
- `main` — stable base
- `demo/drop` — Demo 1: xóa commit debug/console.log
- `demo/edit` — Demo 2: tách 1 commit thành nhiều commit
- `demo/fixup` — Demo 3: gộp nhiều wip commit thành 1

## Getting Started
Clone repo, checkout branch tương ứng để bắt đầu demo.
